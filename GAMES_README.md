# 🎮 Аркадная Гильдия — Игры для фарма XP на фриланс

## Что это?

**Полная система из 5 интерактивных игр** для обучения боту, парсингу, продажам и аналитике через **геймплей**. Каждая игра:
- ✅ Учит практическим навыкам (реальный код)
- ✅ Даёт XP в основной план (синхронизация через localStorage)
- ✅ Красивая графика (Canvas 2D, SVG спрайты)
- ✅ 5-6 уровней с прогрессией

---

## 🎮 Игры в системе

### 1. **🏰 Code Dungeon** (Уже был)
**Тип:** RPG / Квиз-экшен  
**Суть:** Убивай багов-демонов, отвечай на вопросы, вскрывай сундуки

- 4 комнаты (биомы)
- Враги = ошибки кода
- Сундуки = вопросы по плану (парсинг, боты, деплой)
- Спецатака (F-клавиша) с затратой маны
- XP → синхронизируется в главный план

**Запуск:** `games-hub.html` → нажми Code Dungeon

---

### 2. **🤖 Bot Builder Platform** (НОВЫЙ)
**Тип:** Платформер  
**Суть:** Собирай коины кода, прохожди уровни, решай код-челленджи

**5 уровней:**
1. **Основы MVP** → `return "OK"` (простй return)
2. **Error Handling** → try-catch блок
3. **Асинхронный код** → async-await
4. **Regex** → регулярные выражения
5. **MVPComplete** → классы и конструкторы

**Механика:**
- Платформа с врагами 👾 и коинами 🪙
- Нужно собрать N коинов → дойти до флага → решить код для разблока следующего уровня
- Если код верный (regex проверка) → +XP → новый уровень
- Кнопки: WASD, стрелки, русская раскладка (ЦФЫВ)

**Код в textarea:** пишешь решение, нажимаешь "ПРОВЕРИТЬ" → регулярное выражение проверяет шаблон

**XP:** Каждый уровень × награда → сохраняется в localStorage → синхронизируется при загрузке плана

---

### 3. **🛡️ Parser Defense** (НОВЫЙ)
**Тип:** Tower Defense  
**Суть:** Защищайся от волн грязных данных, ставь башни-фильтры

**6 волн:**
1. Грязные данные (нужен Filter)
2. Дубликаты (Set, filter)
3. Быстрый парсинг (Optimize)
4. Большие батчи (Cache)
5. Комбо волна
6. BOSS волна (требует все башни)

**Башни (фильтры кода):**
- 🔍 **Filter** (25 💰) — удаляет шум, средний урон
- ✓ **Validate** (40 💰) — проверка качества, высокий урон
- 💾 **Cache** (60 💰) — кэширует + замедляет, дальний радиус
- ⚡ **Optimize** (80 💰) — самая мощная

**Механика:**
- Враги → "грязные данные" движутся справа налево
- Ты ставишь башни на сетку (клик на Canvas)
- Башни стреляют, убивают врагов → 💰 золото
- Если враг уходит → -1 HP (всего 20 HP)
- Убежден враг → баг в продакшене 💀

**XP:** За каждую убитую волну +XP → сохраняется

---

### 4. **💰 Deal Maker** (НОВЫЙ)
**Тип:** Negotiation / Sales strategy
**Суть:** Закрывай сделки, удерживая баланс цены, доверия и ценности.

- 5 переговорных раундов
- Выбор ответа влияет на доверие, шанс закрытия и итоговый XP
- Учись обосновывать цену, работать с возражениями и продавать без демпинга

**XP:** За удачные ответы и финальное закрытие сделки

---

### 5. **📊 Stats Craft** (НОВЫЙ)
**Тип:** Analytics / Funnel strategy
**Суть:** Анализируй воронки, находи bottleneck и выбирай, что чинить первым.

- 5 сценариев с реальными KPI
- Смотри на конверсии, узкие места и качество лидов
- Учись превращать цифры в конкретные действия

**XP:** За правильные выводы и финальный аналитический план

### XP Синхронизация

**localStorage ключи:**
- `botBuilderXP` ← из Bot Builder Platformer
- `dungeonXP` ← из Code Dungeon
- `parserDefenseXP` ← из Parser Defense

**Главный план (30d-main-tasks.html):**
- Функция `syncGameXP()` вызывается при загрузке
- **Формула:** `(botBuilderXP + dungeonXP + parserDefenseXP) × 0.5` = точки в план
- **Нивелинг:** Каждые 500 очков = +1 уровень

**Пример:**
```
Сыграл в Code Dungeon → 300 XP
Сыграл в Bot Builder → 200 XP
Сыграл в Parser Defense → 100 XP
────────────────────────
Всего: 600 XP × 0.5 = 300 очков в план → 0.6 нового уровня
```
