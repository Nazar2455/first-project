1) Базовый loguru setup

utils/logger.py
from loguru import logger

logger.add("logs/bot.log", rotation="10 MB", retention="7 days")


def info(msg: str):
    logger.info(msg)


def error(msg: str):
    logger.error(msg)

2) Global error handler

пример идеи
try:
    # обработка сообщения
    ...
except Exception as e:
    logger.exception("Unhandled error: {}", e)
    await message.answer("⚠️ Временная ошибка, попробуйте ещё раз")
Пользователь не должен видеть traceback, но ты должен видеть его в логах.

3) Что логировать обязательно

  • Старт/остановка бота
  • Ключевые команды (/start, /admin, /export)
  • Ошибки БД/API
  • Попытки неавторизованного доступа

4) Практика

  1. Подключи loguru.
  2. Добавь global error handler.
  3. Сымитируй 2 ошибки и убедись, что они попали в лог.
  4. Добавь краткий раздел «Troubleshooting» в README.
