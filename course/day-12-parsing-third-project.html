1) План запуска третьего проекта

  1. Выбери новый сайт/категорию.
  2. Собери 3 ключевых селектора.
  3. Подключи экспорт CSV/JSON.
  4. Добавь фильтрацию и краткий отчёт.

2) Универсальный шаблон parser main

main.py
from services.http_client import fetch_html
from parsers.adapter import parse_rows
from services.export import save_csv

URL = "https://example-third-site.com/catalog"


def main() -> None:
    html = fetch_html(URL)
    rows = parse_rows(html)
    save_csv(rows, "result.csv")
    print(f"Parsed: {len(rows)}")


if __name__ == "__main__":
    main()

3) Критерии качества

  • Минимум 100 валидных строк.
  • Нет пустых title/url.
  • Нет массовых дублей.
  • README с описанием полей и запуска.
Третий проект = сильный сигнал для клиента, что ты не «одноразовый» исполнитель.

4) Практика

  1. Сделай третий parser-project.
  2. Добавь его в портфолио с 2 скриншотами.
  3. Сравни результаты трёх парсеров в одной таблице.
  4. Подготовь короткий pitch на 3 предложения.
