1) Каркас README для парсера

README.md
# Parser Project

## Что делает
Собирает карточки товаров/объявлений с сайта и сохраняет в CSV/JSON.

## Поля
- title
- price
- url
- updated_at

## Фильтры
- min_price
- max_price

## Быстрый запуск
1. pip install -r requirements.txt
2. python main.py

## Результат
Файлы result.csv и result.json для анализа и отчётов.

2) Как показать результат клиенту

  1. Скриншот исходного сайта (что собираем).
  2. Скриншот CSV/Google Sheets результата.
  3. Короткое видео «запуск → итог».
  4. Таблица «до/после» по времени ручной работы.
Показывай не код, а бизнес-результат: «автоматически обновляем 300+ позиций за 2 минуты».
