Тема: выгрузка результатов парсинга в Google Sheets, чтобы клиент видел обновления без техподготовки.

1) Стратегия выгрузки

  • append mode: добавляем новые строки в конец.
  • replace mode: очищаем лист и пишем свежий срез.
  • upsert mode: обновляем по ключу (URL/ID).

2) Пример append в таблицу

services/sheets_export.py
import gspread
from google.oauth2.service_account import Credentials

SCOPES = ["https://www.googleapis.com/auth/spreadsheets"]


def get_ws():
    creds = Credentials.from_service_account_file("service-account.json", scopes=SCOPES)
    client = gspread.authorize(creds)
    return client.open("Parser Reports").worksheet("items")


def append_items(rows: list[dict]):
    ws = get_ws()
    values = []
    for r in rows:
        values.append([r.get("title", ""), r.get("price", ""), r.get("url", "")])
    if values:
        ws.append_rows(values)

3) Как не плодить дубли

  1. Используй URL как уникальный ключ.
  2. Перед записью проверяй, есть ли URL в листе.
  3. Либо храни snapshot в JSON и выгружай только новые.
Клиенты ценят чистые таблицы без дублей и мусора.

4) Практика

  1. Выгрузи минимум 50 строк в Google Sheets.
  2. Сделай повторный запуск без дублей.
  3. Добавь столбец с датой обновления.
  4. Покажи таблицу как часть портфолио-кейса.
