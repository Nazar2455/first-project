1) Подготовка service account

  1. Создай проект в Google Cloud.
  2. Включи Google Sheets API.
  3. Создай Service Account и JSON-ключ.
  4. Поделись таблицей с email сервисного аккаунта.

2) Код записи лида в таблицу

services/google_sheets.py
import gspread
from google.oauth2.service_account import Credentials
from datetime import datetime

SCOPES = ["https://www.googleapis.com/auth/spreadsheets"]


def get_sheet():
    creds = Credentials.from_service_account_file("service-account.json", scopes=SCOPES)
    client = gspread.authorize(creds)
    return client.open("Leads CRM").worksheet("leads")


def save_lead_to_sheets(name: str, phone: str, comment: str):
    ws = get_sheet()
    ws.append_row([
        datetime.now().strftime("%Y-%m-%d %H:%M:%S"),
        name,
        phone,
        comment
    ])
В проде ключ лучше хранить вне репозитория и задавать путь через переменную окружения.

3) Интеграция в FSM-финал

После сохранения в SQLite вызывай save_lead_to_sheets(...). Если Sheets временно недоступен — не роняй бота, логируй ошибку и отвечай пользователю штатно.
